# 台股兵法智謀全媒體觀測系統

這是依照你提供的最新版程式碼整理出的 **GitHub 可用 React + Vite + TypeScript 專案**。

## 本機執行
```bash
npm install
npm run dev
```

## 建置
```bash
npm run build
```

## 適用情境
- 上傳到 GitHub 管理原始碼
- 匯入 Vercel / Netlify 部署
- 也可再接 GitHub Pages

## 專案結構
- `src/App.tsx`：主程式
- `src/main.tsx`：React 入口
- `src/index.css`：基本樣式
- `index.html`：Vite 首頁
- `vite.config.ts`：部署設定
